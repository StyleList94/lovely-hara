# styleli.sh

색에 빠지고 치수를 재고 끈을 매만지고 떨림을 느껴보고 맵시나게.

## 소재

- 분위기 잡아주는 조명 (Astro)
- 터치할수록 반응하는 소재 (React 19)
- 몸매 살려주는 핏 (Tailwind CSS 4)
- 밤새 돌아가는 작업실 (Cloudflare Workers)

## 재단법

```bash
bun run build       # 옷 완성
bun run preview     # 거울 앞에서 한번 입어봄
bun run deploy      # 행거에 걸기
bun run lint        # 삐져나온 실밥 정리
bun run test        # 세탁기 돌려도 안 줄어드는지 확인
```

## 피팅룸

```text
src/
├── features/       # 입어볼 옷들
├── shared/         # 공용 액세서리
├── pages/          # 거울
├── layouts/        # 커튼
└── actions/        # 커튼 뒤에서 일어나는 일
```
